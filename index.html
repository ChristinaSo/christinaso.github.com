W3Lc0m3, cyber visitor...

W3Lc0m3, cyber visitor. I am an intern at Youth Radio, a non-profit organization in Oakland. In the Mobile Action Lab, a group I am part of, we are currently learning programming languages, such as HTML, CSS, Javascript, Python, and so on. You learn so much at Youth Radio!

So far, in the Mobile Action Lab, we have created two applications that can be accessed using a computer or smart phone. One of the apps, Forage City, has already been soft launched at Code for America, a hackathon.

The second app, which I will not disclose until it has been launched, is coming along extremely well.

Although the lab team has created two apps, neither of the two apps were programmed by the youth in the team. Lacking a teacher to teach us, we sought the help of programmers outside of Youth Radio to help us create those two apps.

Fortunately, this year (2012) we were able to learn how to code with the help of our newly hired teacher, Kurt. Our journey to the programming world has begun!
